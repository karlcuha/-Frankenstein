Чат-бот

Адаптивный чат-бот на JavaScript. Работает прямо в браузере, без сервера.


Запуск

1 Откройте папку проекта.
2 Запустите файл "index.html" в браузере.



 Файлы

```text
assets/      - картинки и иконка
index.html   - разметка страницы
styles.css   - стили чата
script.js    - логика чат-бота
README.md    - инструкция
```


Команды

```text
/start
/name: Вася
/number: 7, 9
+
-
*
/
/stop
```


 Пример

```text
/start
/name: Вася
/number: 7, 9
+
```


Ответ:

```text
Результат: 16
```
